## LAB-1: Ortam Kurulumu · Terminal · Git Akisi · Modern Web Projesi

### Hakkinda
Bu proje, Web Tasarimi ve Programlama dersi LAB-1 kapsaminda
Vite + React + TypeScript kullanilarak olusturulmustur.

### Kullanilan Teknolojiler
- **React** 18
- **TypeScript**
- **Vite** 6
- **Node.js** (LTS)
- **Git** & **GitHub**

### Kurulum
```bash
npm install
```

### Calistirma
```bash
npm run dev
```
Tarayicida `http://localhost:5173` adresini ac.

### Build
```bash
npm run build
```

## LAB-2: Semantik HTML5 · Erisebilirlik (a11y) · Form Temelleri

### Hakkinda
LAB-2 kapsaminda semantik HTML5, erisebilirlik ilkeleri ve form
yapilari uygulanarak kisisel portfolyo sayfasi olusturulmustur.

### Uygulanan Konular
- Semantik HTML5 etiketleri (`header`, `nav`, `main`, `section`, `article`, `footer`)
- Heading hiyerarsisi (h1 → h2 → h3)
- Alt metin (alt attribute) ve figure/figcaption kullanimi
- ARIA oznitelikleri (`aria-label`, `aria-describedby`, `role="alert"`)
- Tab ile gezinme ve focus gostergesi
- Skip navigation baglantisi
- Form dogrulama (`required`, `minlength`, `pattern`, `type`)
- Hata mesaji alanlari (`role="alert"`)
- Lighthouse erisebilirlik testi (hedef: 90+ puan)
